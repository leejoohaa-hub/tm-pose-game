# GAME_RULE.md - Catch Zone

## 1. 개요
**게임 이름**: Catch Zone  
**설명**: 화면 상단에서 떨어지는 과일을 바구니로 받아 점수를 획득하고, 폭탄을 피해 최대한 오래 살아남는 게임입니다. 포즈 인식을 통해 바구니를 좌/중/우로 조작합니다.

---

## 2. 구역 정의
화면은 가로로 3개의 구역으로 나뉩니다.
- **LEFT (왼쪽)**: 화면의 왼쪽 1/3 영역
- **CENTER (가운데)**: 화면의 중앙 1/3 영역
- **RIGHT (오른쪽)**: 화면의 오른쪽 1/3 영역

---

## 3. 조작 방식
플레이어의 포즈에 따라 바구니(Basket)가 해당 구역으로 이동합니다.
- **왼쪽 포즈 (Head Tilt Left 등)** → 바구니가 **LEFT** 구역으로 이동
- **중립 포즈 (Straight)** → 바구니가 **CENTER** 구역으로 이동
- **오른쪽 포즈 (Head Tilt Right 등)** → 바구니가 **RIGHT** 구역으로 이동

## 5. 게임 규칙

### 기본 진행
- 아이템은 화면 상단에서 랜덤한 구역(Left, Center, Right)으로 떨어집니다.
- 바구니가 아이템이 떨어지는 구역에 위치해야 획득 판정이 납니다.

### 점수 및 획득
- **과일 획득**: 해당 과일의 점수만큼 Total Score 증가
- **폭탄 획득**: 게임 종료 (Game Over)

### 놓침 (Miss) 패널티
- **과일**을 바구니로 받지 못하고 바닥으로 흘려보낸 경우:
  - **1회 놓침**: 경고 (UI 표시)
  - **2회 놓침**: **게임 오버**
- 폭탄은 놓쳐도 아무런 패널티가 없습니다. (오히려 피해야 함)

---

## 6. 레벨 시스템 (Level System)
게임은 **20초 단위**의 단계(Level)로 구성됩니다. 시간이 지날수록 난이도가 상승합니다.

- **진행 방식**: 게임 오버 없이 20초를 버티면 자동으로 다음 레벨로 넘어갑니다.
- **Drop Time (낙하 속도)**:
  - **Level 1**: 2.0초 (화면 상단에서 바닥까지 닿는 시간)
  - **Level Up**: 단계마다 0.2초씩 감소 (점점 빨라짐)
- **Spawn Interval (생성 간격)**:
  - 현재 Drop Time의 **60% ~ 80%** 사이 랜덤 간격으로 아이템 생성

---

## 7. UI 요소
화면에 표시되어야 할 정보:
1. **Score**: 현재 점수
2. **Level**: 현재 단계
3. **Time**: 다음 레벨까지 남은 시간 (혹은 전체 생존 시간)
4. **Life / Warning**: 놓친 횟수 표시 (예: 하트 2개 또는 경고 문구)
5. **Game Over 화면**: 최종 점수 및 재시작 버튼 포함
